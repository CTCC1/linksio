# 【国服头像】G5 OU比赛

http://china.psim.us/ 

国服已经举办了一系列比赛，产生了多位获得专属头像的冠军选手。作为明年中国队出征世界杯的准备，将举办G5 OU的比赛给大家提供训练机会。

## 报名

加入国服比赛QQ群 106091896，表明参加G5比赛即可。

## 规则

比赛赛制待定（小组循环赛4进2+单淘汰？）

淘汰阶段三局两胜。请保存回放在赛后发给主办方。（可以在挑战前在公屏输入/ionext防止其他比赛对手围观）

采用PS GEN5 OU规则，具体内容如下：

```

```

## 奖励

如果报名总人数在64人以上，则积分赛最后的冠亚军将获得国服头像。参赛人数达32人时冠军获得国服头像。

如果已经拥有国服头像，则可以选择一只在所有轮次中自己使用过的精灵显示在名字前。
